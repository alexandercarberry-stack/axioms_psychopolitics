# The Axioms of Psychopolitics

This repository contains the **canonical v1.0 text** of *The Axioms of Psychopolitics*.

The work presents a **closed axiomatic framework** for analysing politics as a libidinal and symbolic theatre, examining fantasy, coherence, carcerality, race, technology, and power as structuring forces of political life.

This text is **doctrinal rather than explanatory**. It is intended to be cited, inhabited, and applied, not summarised or defended.

For citation and archival reference, see the associated **DOI record**.
